# Tank2017
一个基于MFC/WIN32写的坦克游戏，尝试用最基础的知识写一些东西，仅为了学习的目的

## 说明
代码很粗糙，功能也非常之简单，仅做了些基本的游戏运行逻辑，之后会定期添加功能

## 游戏操作
【J】射击，【WSAD】上下左右，【ESC】暂停游戏
